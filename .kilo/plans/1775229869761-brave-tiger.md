# Plan: Offset Pipeline — offsets/ → build → cache_offsets/ → Runtime

## Architecture Overview

```
cs2overlay/
├── build.bat
├── offsets/                         ← папка проекта (рядом с build.bat)
│   └── output/                      ← сюда кидаешь папку output из дампера
│       ├── client_dll.json
│       ├── client_dll.hpp
│       ├── offsets.json
│       ├── offsets.hpp
│       └── ...
├── build/
│   └── Release/
│       ├── cs2overlay.exe
│       └── cache_offsets/           ← создаётся при билде / при GitHub update
│           ├── offsets.json
│           └── client_dll.json
```

## Поток данных

1. **Build-time**: `build.bat` ищет `offsets/output/` → копирует нужные файлы в `build/Release/cache_offsets/`
2. **Runtime**: exe читает `cache_offsets/` рядом с собой → парсит JSON или HPP → применяет к `SDK::Offsets`
3. **GitHub fallback**: если `cache_offsets/` нет или невалиден → скачивает с GitHub → сохраняет в `cache_offsets/`
4. **UI Update**: кнопка "Update Offsets" → GitHub → перезапись `cache_offsets/`
5. **UI Reload**: кнопка "Reload Offsets" → перечитать `cache_offsets/` без скачивания

## Step 1: `build.bat` — Copy offsets from offsets/output/ to build output

**Before cmake configure**, add:
- Check if `offsets\output\` exists
- If yes:
  - Create `build\Release\cache_offsets\` directory
  - Copy `offsets\output\offsets.json` → `build\Release\cache_offsets\offsets.json`
  - Copy `offsets\output\client_dll.json` → `build\Release\cache_offsets\client_dll.json`
  - If JSON not found, try `.hpp`: copy `offsets\output\client_dll.hpp` → `cache_offsets\client_dll.hpp`
  - Print `[OFFSET] Copied offsets from offsets/output/ to cache_offsets/`
- If no: print `[OFFSET] No offsets/output/ found, skipping`
- Continue to cmake build as normal

## Step 2: New file `src/core/sdk/offset_file_loader.h/cpp`

**Responsibility**: Find and load raw offset files (JSON or HPP) from `cache_offsets/` directory next to the exe.

```cpp
class OffsetFileLoader {
public:
    struct FileResult {
        std::string offsetsContent;    // JSON or empty
        std::string clientContent;     // JSON or empty
        std::string offsetsHpp;        // HPP or empty
        std::string clientHpp;         // HPP or empty
        bool hasJson() const;
        bool hasHpp() const;
    };

    FileResult LoadFromCacheDir();
    FileResult DownloadFromGitHub();
    void SaveToCacheDir(const std::string& offsetsJson, const std::string& clientJson);
};
```

**Logic:**
- `LoadFromCacheDir()`:
  - Get exe directory → look for `cache_offsets/` subfolder
  - Try to read `offsets.json` + `client_dll.json`
  - If JSON not found, try `client_dll.hpp`
  - Return whatever was found (may be partial)
- `DownloadFromGitHub()`:
  - Fetch `offsets.json` + `client_dll.json` from a2x/cs2-dumper
  - Return content strings
- `SaveToCacheDir()`:
  - Create `cache_offsets/` if not exists
  - Write both files

## Step 3: New file `src/core/sdk/offset_parser.h/cpp`

**Responsibility**: Parse raw file content (JSON or HPP) into a flat `std::unordered_map<std::string, ptrdiff_t>`.

```cpp
class OffsetParser {
public:
    struct ParsedOffsets {
        ptrdiff_t dwEntityList = 0;
        ptrdiff_t dwLocalPlayerPawn = 0;
        ptrdiff_t dwLocalPlayerController = 0;
        ptrdiff_t dwViewMatrix = 0;
        ptrdiff_t dwPlantedC4 = 0;
        ptrdiff_t m_fFlags = 0;
        ptrdiff_t m_vecVelocity = 0;
        ptrdiff_t m_iHealth = 0;
        ptrdiff_t m_iTeamNum = 0;
        ptrdiff_t m_vOldOrigin = 0;
        ptrdiff_t m_pGameSceneNode = 0;
        ptrdiff_t m_modelState = 0;
        ptrdiff_t m_hPlayerPawn = 0;
        ptrdiff_t m_iszPlayerName = 0;
        ptrdiff_t m_pClippingWeapon = 0;
        ptrdiff_t m_vecViewOffset = 0;
        ptrdiff_t m_angEyeAngles = 0;
        ptrdiff_t m_aimPunchAngle = 0;
        ptrdiff_t m_iIDEntIndex = 0;
        ptrdiff_t m_bIsScoped = 0;
        ptrdiff_t m_iShotsFired = 0;
        ptrdiff_t m_nBombSite = 0;
        ptrdiff_t m_bBombTicking = 0;
        ptrdiff_t m_flTimerLength = 0;
        int missingCount() const;  // count of fields that are still 0
    };

    ParsedOffsets ParseJson(const std::string& offsetsJson, const std::string& clientJson);
    ParsedOffsets ParseHpp(const std::string& offsetsHpp, const std::string& clientHpp);
    ParsedOffsets Parse(const OffsetFileLoader::FileResult& files);
};
```

**JSON parsing** (existing logic, extracted):
- `offsets.json`: read `client.dll` → `dwEntityList`, `dwLocalPlayerPawn`, etc.
- `client_dll.json`: iterate `client.dll.classes` → find fields by name in `C_BaseEntity`, `C_CSPlayerPawn`, `C_CSPlayerController`, `C_PlantedC4`

**HPP parsing** (new):
- Use regex to match `#define\s+(\w+)\s+(0x[0-9a-fA-F]+|\d+)`
- Map known macro names to fields:
  - `offsets.hpp`: `#define client_dwEntityList 0x...` → `dwEntityList`
  - `client_dll.hpp`: `#define CBaseEntity_m_iHealth 0x...` → `m_iHealth`
- Known class prefixes: `CBaseEntity_`, `CCSPlayerPawn_`, `CCSPlayerController_`, `CPlantedC4_`

**Priority**: If both JSON and HPP available → use JSON (more reliable). If only HPP → parse HPP.

## Step 4: New file `src/core/sdk/offset_applier.h/cpp`

**Responsibility**: Apply parsed offsets to `SDK::Offsets` + validate + log.

```cpp
class OffsetApplier {
public:
    void Apply(const OffsetParser::ParsedOffsets& parsed);
    bool Validate() const;  // checks critical fields != 0
    void LogStatus() const;
};
```

**Logic:**
- Copy each field from `ParsedOffsets` → `SDK::Offsets`
- Validate: `dwEntityList`, `dwLocalPlayerPawn`, `dwViewMatrix` must be != 0
- Log all values to console with `[+]` prefix
- Log warnings for any field that is 0 (missing from dump)

## Step 5: Rewrite `offset_loader.h/cpp` — Facade

```cpp
class OffsetLoader {
public:
    void LoadOffsets();           // Main entry: FileLoader → Parser → Applier
    void ReloadOffsets();         // Re-read cache_offsets/ without GitHub
    void ForceUpdateFromGitHub(); // Download → Save → Parse → Apply

private:
    OffsetFileLoader fileLoader;
    OffsetParser parser;
    OffsetApplier applier;
};
```

**`LoadOffsets()` flow:**
1. `fileLoader.LoadFromCacheDir()` → try `cache_offsets/`
2. If valid files found → `parser.Parse()` → `applier.Apply()` → done
3. If no valid files → `fileLoader.DownloadFromGitHub()` → `fileLoader.SaveToCacheDir()` → `parser.Parse()` → `applier.Apply()`

**`ReloadOffsets()` flow:**
1. `fileLoader.LoadFromCacheDir()` → `parser.Parse()` → `applier.Apply()`
2. No GitHub, no save — just re-read what's on disk

**`ForceUpdateFromGitHub()` flow:**
1. `fileLoader.DownloadFromGitHub()` → `fileLoader.SaveToCacheDir()` → `parser.Parse()` → `applier.Apply()`
