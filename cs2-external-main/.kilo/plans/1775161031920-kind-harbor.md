# План исправления ESP в CS2 Overlay

## Корневая причина неработающего ESP

**Старая версия** (`cs2-external-main`) использовала **regex-парсинг** сырой JSON-строки:
```cpp
std::regex re("\"" + key + "\"\\s*:\\s*(\\d+)");
std::regex_search(src, m, re);
```
Regex ищет `"dwEntityList": 38470232` где угодно в файле — вложенность не важна.

**Новая версия** перешла на `nlohmann/json`, но неправильно навигирует по структуре:
- `offsets.json` имеет структуру: `{"client.dll": {"dwEntityList": 38470232, ...}}`
- Код искал: `getVal(offsetsJson, "dwEntityList")` → **НЕ НАХОДИТ** (ключ вложен в `"client.dll"`)
- Результат: `dwEntityList = 0`, `dwViewMatrix = 0`, `dwLocalPlayerPawn = 0` → ESP не может читать память

## Исправления

### 1. offset_loader.cpp — навигация по вложенной структуре offsets.json
**Файл:** `src/core/sdk/offset_loader.cpp`

**Проблема:** `ApplyOffsetsFromJson` ищет engine-оффсеты на верхнем уровне JSON, но они вложены в `"client.dll"`.

**Исправление:** Заменить:
```cpp
Offsets::dwEntityList = getVal(offsetsJson, "dwEntityList");
```
На:
```cpp
if (offsetsJson.contains("client.dll")) {
    const auto& client = offsetsJson["client.dll"];
    Offsets::dwEntityList = getVal(client, "dwEntityList");
    Offsets::dwLocalPlayerPawn = getVal(client, "dwLocalPlayerPawn");
    Offsets::dwLocalPlayerController = getVal(client, "dwLocalPlayerController");
    Offsets::dwViewMatrix = getVal(client, "dwViewMatrix");
    Offsets::dwPlantedC4 = getVal(client, "dwPlantedC4");
}
```

**Статус:** ✅ Уже исправлено в предыдущей итерации

### 2. offset_loader.cpp — FindFieldInClientJson для member offsets
**Файл:** `src/core/sdk/offset_loader.cpp`

**Проблема:** `client_dll.json` имеет структуру `{"client.dll": {"classes": {"CCSPlayerPawn": {"fields": {"m_iHealth": 852}}}}}`. Нужно искать по всем классам.

**Исправление:** Функция `FindFieldInClientJson` проходит по всем классам и их полям.

**Статус:** ✅ Уже исправлено в предыдущей итерации

### 3. Удаление кэша
**Файлы:** `offsets_cache.json`, `client_cache.json`

**Проблема:** Старый кэш может содержать данные в неправильном формате.

**Исправление:** Удалить файлы кэша, чтобы при запуске загрузились свежие оффсеты.

**Статус:** ✅ Уже выполнено
